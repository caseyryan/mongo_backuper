RESTful сервис для создания резервных копий MongoDB на сервере. 
Но фактически может делать по расписанию архивы из любых доступных папок и складывать их
либо локально, либо заливать на гугл или яндекс диск. Или все варианты одновременно
Пути к нужны папкам нужно заранее указать в переменных среды
